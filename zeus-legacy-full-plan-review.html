Executive Overview

All Todo
Critical Blocker: Spine Version Incompatibility
Zeus uses Spine 3.8.99 format. The editor uses @esotericsoftware/spine-pixi-v8 (Spine 4.x). Phase 0.1 must resolve this before Phase 7 (Character & Animations) can begin.
Key Design Decision
Zeus Legacy is a completely new component type (zeus-legacy), NOT reusing the existing cascade-slot component. Every UI element becomes a separate child GameObject, individually editable. The root config contains ONLY game-level settings.
